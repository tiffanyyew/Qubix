HTML Learning - Level 6

Level 6 - Advanced HTML

1. Which HTML element is used to define a form for user input?

!
Hint: Forms are used to collect user data, and the element is typically used with `input` tags.
Hover for answer
<form>

2. Which HTML element is used to embed sound or audio files?

!
Hint: This element allows embedding of media content like sound or music files.
Hover for answer
<audio>

3. Which HTML5 element is used to define a section of content within a document?

!
Hint: This semantic element is used to group content logically, often with a heading.
Hover for answer
<section>

4. Which HTML element is used to define a label for an input element?

!
Hint: This element associates a description with a form control.
Hover for answer
<label>

5. Which HTML element is used to define navigation links?

!
Hint: This element groups a set of navigation links, helping users find their way around the website.
Hover for answer
<nav>
