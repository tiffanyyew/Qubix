CSS Learning - Level 6

Level 6 - Intermediate CSS Concepts

1. Write CSS to set the background color of an element to red.

!
Hint: Use the `background-color` property.
Hover for answer
#element { background-color: red; }

2. Write CSS to create a 100x100 px box with a blue background.

!
Hint: Use `width`, `height`, and `background-color`.
Hover for answer
.box { width: 100px; height: 100px; background-color: blue; }

3. Write CSS to set the font size of a paragraph to 16px and the color to green.

!
Hint: Use `font-size` and `color` properties.
Hover for answer
p { font-size: 16px; color: green; }

4. Write CSS to create a flex container with items spaced equally across the width.

!
Hint: Use `display: flex` and `justify-content: space-between`.
Hover for answer
.container { display: flex; justify-content: space-between; }

5. Write a media query to change the font size of elements with class `.responsive` to 12px on screens smaller than 600px.

!
Hint: Use `@media` with `max-width` to target smaller screens.
Hover for answer
@media (max-width: 600px) { .responsive { font-size: 12px; } }
