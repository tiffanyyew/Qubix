CSS Learning - Level 7

Level 7 - Advanced CSS Techniques

1. What CSS pseudo-element is used to insert content after an element?

!
Hint: This pseudo-element is often used to add decorative content.
Hover for answer
::after

2. What CSS property and value are used to make an element stay in place while scrolling?

!
Hint: Think about how navigation bars remain visible.
Hover for answer
position: fixed;

3. Write a CSS media query for screens smaller than 600px wide.

!
Hint: Use `@media` with a `max-width` condition.
Hover for answer
@media (max-width: 600px)

4. Write a CSS rule to create a grid with three equal columns.

!
Hint: Use the `grid-template-columns` property.
Hover for answer
grid-template-columns: repeat(3, 1fr);

5. What CSS property controls the stack order of elements?

!
Hint: Higher values make elements appear on top.
Hover for answer
z-index: 1000;
