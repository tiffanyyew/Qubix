Python Learning - Level 7

Level 7 - Lists and Dictionaries

1. Create a list with the numbers 1, 2, 3, and 4.

!
Hint: Use square brackets `[]` to create a list.
Hover for answer
my_list = [1, 2, 3, 4]

2. Create a dictionary with keys 'name' and 'age'.

!
Hint: Use curly braces `{}` to create a dictionary.
Hover for answer
my_dict = {'name': 'John', 'age': 30}

3. Add the number 5 to the list you created.

!
Hint: Use the `append()` method to add items to a list.
Hover for answer
my_list.append(5)

4. Update the age in the dictionary to 31.

!
Hint: Use the dictionary key to update its value.
Hover for answer
my_dict['age'] = 31

5. Remove the first element from the list.

!
Hint: Use the `del` keyword to delete an item by index.
Hover for answer
del my_list[0]
