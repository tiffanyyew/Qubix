HTML Learning - Level 7

Level 7 - Advanced HTML and Accessibility

1. Which attribute is used to describe an image for accessibility purposes?

!
Hint: This attribute provides a description of the image for screen readers.
Hover for answer
alt

2. Which attribute is used to define the role of an element for screen readers?

!
Hint: This attribute helps screen readers understand the purpose of an element.
Hover for answer
role

3. How do you associate a label with a specific form input element?

!
Hint: The `for` attribute in the label tag is used to associate it with an input element by matching the input's `id`.
Hover for answer
<label for="inputId">

4. Which HTML element is used to create a clickable button?

!
Hint: This element is used to trigger actions like submitting a form or running a script.
Hover for answer
<button>

5. Which HTML element is used to embed an external resource such as a webpage?

!
Hint: This element is used to embed another HTML page or other media within a webpage.
Hover for answer
<iframe>
