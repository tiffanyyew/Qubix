Python Learning - Level 6

Level 6 - Functions and Loops

1. Write a function that takes a name and returns a greeting.

!
Hint: Use `def` to define the function and `return` for output.
Hover for answer
def greet(name): return f'Hello, {name}'

2. Write a loop to print numbers from 0 to 4.

!
Hint: Use a `for` loop with `range()`.
Hover for answer
for i in range(5): print(i)

3. Write a function to add two numbers and return the result.

!
Hint: Use two parameters and a `return` statement.
Hover for answer
def add(a, b): return a + b

4. Write a `while` loop to increment a variable `x` until it is 5.

!
Hint: Use a condition to keep looping until `x` reaches 5.
Hover for answer
while x < 5: x += 1

5. Write an if-else statement to check if a number is even or odd.

!
Hint: Use the modulus operator `%` to check divisibility by 2.
Hover for answer
if n % 2 == 0: print('Even') else: print('Odd')
