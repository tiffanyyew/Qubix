Python Learning - Level 8

Level 8 - File I/O and Error Handling

1. Write code to create a file 'file.txt' and write 'Hello, World!' in it.

!
Hint: Use `open()` with `'w'` mode to write to a file.
Hover for answer
with open('file.txt', 'w') as f: f.write('Hello, World!')

2. Write code that handles a division by zero error.

!
Hint: Use `try` and `except` to handle exceptions.
Hover for answer
try: x = 1 / 0 except ZeroDivisionError: print('Error!')

3. Write code to read the content of 'file.txt'.

!
Hint: Use `open()` with `'r'` mode to read a file.
Hover for answer
with open('file.txt', 'r') as f: print(f.read())

4. Write a `finally` block to print a message.

!
Hint: Use `finally` to execute code after a `try-except` block.
Hover for answer
finally: print('This runs no matter what')

5. Write code to append 'Goodbye!' to 'file.txt'.

!
Hint: Use `'a'` mode to append to a file.
Hover for answer
with open('file.txt', 'a') as f: f.write('Goodbye!')
